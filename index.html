Position

En HTML, el atributo position se refiere a una propiedad CSS que controla cómo un elemento se posiciona dentro de su contenedor o del documento. Hay varios valores posibles para esta propiedad, y cada uno afecta de manera diferente el comportamiento del elemento en la página. Estos son los valores más comunes de position:

position: relative;

position: static;

position: absolute;

position: sticky;

Display

En HTML, la propiedad display es una de las más importantes de CSS y controla cómo se muestra un elemento en la página. Define el comportamiento del cuadro de un elemento y cómo interactúa con otros elementos. Hay varios valores de display que se pueden utilizar, y cada uno afecta de manera diferente al diseño y estructura del contenido. A continuación, se presentan los valores más comunes de la propiedad display:

Display: block;

Display: inline;

Display: inline-block;

Display: none;

Display: flex;

Display: grid;

Display: inherit;

Animacion

La animación en HTML se refiere a la capacidad de mover, cambiar o transformar elementos en una página web de manera dinámica, proporcionando interactividad y efectos visuales. Aunque HTML por sí solo no puede crear animaciones, se combina con CSS (Cascading Style Sheets) y JavaScript para lograrlo.

Nuevas Propiedades de css3

CSS3 introdujo muchas propiedades nuevas que mejoraron significativamente las capacidades del diseño web, haciéndolo más flexible, moderno y dinámico. Algunas de las propiedades más importantes de CSS3 incluyen:

box-radius

box-shadow

text-shadow

background linear-gradient

background radial-gradient
